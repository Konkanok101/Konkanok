ความหมายของ While Loop

พื้นฐานและการใช้งาน While Loop ในภาษาซี

หน้าแรก ความหมายของ While Loop การทำงานของลูป While บทเรียนที่ 2 ติดต่อเรา

While Loop คืออะไร?

While Loop เป็นโครงสร้างควบคุมการทำงานแบบวนซ้ำในภาษาซี โดยจะทำงานซ้ำตามเงื่อนไขที่กำหนดจนกว่าเงื่อนไขจะเป็นเท็จ (false)

โครงสร้าง:

while (เงื่อนไข) { // คำสั่งที่ต้องการให้ทำซ้ำ }

ตัวอย่างการใช้งาน While Loop

#include <stdio.h> int main() { int i = 1; while (i <= 5) { printf("รอบที่ %d\n", i); i++; } return 0; }

ประโยชน์ของ While Loop

  • ช่วยให้เขียนโค้ดที่ทำงานซ้ำได้อย่างง่ายดาย
  • เหมาะสำหรับสถานการณ์ที่ไม่สามารถคาดเดาจำนวนรอบล่วงหน้าได้
  • ปรับปรุงประสิทธิภาพการทำงานของโปรแกรม

ข้อควรระวัง

ควรตรวจสอบให้แน่ใจว่าเงื่อนไขใน While Loop จะเปลี่ยนแปลงได้ เพื่อหลีกเลี่ยงการเกิดลูปที่ไม่มีวันสิ้นสุด (infinite loop)

ตัวอย่างเพิ่มเติม

การคำนวณผลรวมของตัวเลข 1 ถึง 10:

#include <stdio.h> int main() { int sum = 0, i = 1; while (i <= 10) { sum += i; i++; } printf("ผลรวมคือ %d\n", sum); return 0; }
