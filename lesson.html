เรียนรู้การทำงานของลูป While ในภาษาซี

หน้าแรก ความหมายของ While Loop การทำงานของลูป While บทเรียนที่ 2 ติดต่อเรา

While Loop ในภาษา C

ลูป While ใช้สำหรับการทำงานซ้ำจนกว่าจะมีเงื่อนไขเป็นเท็จ (false) โดยรูปแบบพื้นฐานของคำสั่ง While ในภาษา C มีดังนี้:

while (เงื่อนไข) {
    // โค้ดที่ต้องการให้ทำซ้ำ
}

ตัวอย่างการใช้งาน

#include <stdio.h>

int main() {
    int count = 1;
    while (count <= 5) {
        printf("นับ: %d\n", count);
        count++;
    }
    return 0;
}

  • นับ: 4
  • นับ: 5

คำถามทดสอบ

1. เขียนโปรแกรมที่ใช้ลูป While แสดงตัวเลขตั้งแต่ 1 ถึง 10

2. เขียนโปรแกรมที่ใช้ลูป While แสดงเฉพาะตัวเลขคู่ระหว่าง 1 ถึง 20
